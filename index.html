Research
Compressed neurons: Compression based memory management

Current memory management techniques for machine learning workloads include swapping and recomputating memory. Created a policy that can perform compression as an alternative.

• Benchmarked multiple compression libraries - nvcomp and zfp and are using superneurons as the baseline ; • Preliminary results show compression is atleast 2x faster than swapping.

Automatic Thread Scheduler

Proved with benchmarks the eﬀect of scheduling on thread co-degradations.

Created a scheduler that automatically learns the groups of threads to schedule based on assigned rewards.

The scheduler has 10% higher throughput as compared to the CFS by setting aﬃnity of threads.

sThreads: Thread scheduling for reducing tail latency

Created a benchmarking suite, benchmark execution framework and pre-emptive user space threading library.

In a non-preemptive FCFS model a long running process will block shorter processes, raising tail latency.

Memory intensive threads have a signiﬁcant impact (20-30%) on the tail latency.
